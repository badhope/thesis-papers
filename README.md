# ☢️ Nuclear Weapon Simulator

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/version-2.0.0-green.svg)]()
[![Platform](https://img.shields.io/badge/platform-Web-orange.svg)]()

一个基于物理模型的核武器效应模拟器，用于教育和研究目的。通过可视化世界地图，模拟核爆炸的各种效应，包括冲击波、热辐射、电离辐射等，并估算人员伤亡和基础设施损毁情况。

> ⚠️ **免责声明**：本模拟器仅供教育和研究目的使用。所有计算基于公开的物理模型和历史数据，结果仅供参考，不构成任何实际决策依据。

## ✨ 功能特性

### 🗺️ 交互式地图
- **详细世界地图**：70+ 国家/地区轮廓，支持缩放、拖拽
- **城市标注**：60+ 世界主要城市，显示人口和坐标
- **目标选择**：点击地图或搜索城市选择目标
- **实时可视化**：爆炸影响范围动态渲染

### 💥 核武器模拟
- **预设武器库**：10+ 真实核武器型号
  - 历史武器：小男孩 (15kt)、胖子 (21kt)、沙皇炸弹 (50Mt)
  - 现役武器：W76、W87、B83、东风-41、布拉瓦等
- **自定义当量**：支持 1kt - 100Mt 自定义
- **爆炸高度**：地面爆炸、空中爆炸、高空爆炸

### 📊 效应计算
- **火球半径**：物质完全气化区域
- **冲击波**：重度/中度/轻度破坏区 (20/5/2 psi)
- **热辐射**：三度烧伤范围
- **电离辐射**：500 rem 辐射区
- **电磁脉冲 (EMP)**：电子设备影响范围

### 👥 伤亡估算
- **人口密度**：可调整目标区域人口密度
- **时间段影响**：夜间/白天/高峰时段
- **国家特色数据**：80+ 国家的医疗、避难、建筑数据
- **建筑类型**：8种建筑类型的抗爆/防火能力

### 📈 长期影响
- **经济损失**：直接/间接经济损失估算
- **恢复时间**：基础设施恢复预测
- **辐射落尘**：根据气候条件的扩散模型

## 🚀 快速开始

### 环境要求

- 现代浏览器 (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)
- 本地 Web 服务器 (可选，用于本地开发)

### 安装运行

```bash
# 克隆仓库
git clone https://github.com/yourusername/bumb.git
cd bumb

# 方法1: Python 简易服务器
python -m http.server 8080

# 方法2: Node.js 服务器
npx serve

# 方法3: PHP 内置服务器
php -S localhost:8080

# 打开浏览器访问
# http://localhost:8080
```

### 使用说明

1. **选择武器**：从下拉菜单选择预设武器或自定义当量
2. **选择目标**：点击地图、搜索城市或从列表选择
3. **调整参数**：设置人口密度、时间段等
4. **开始模拟**：点击"开始模拟"按钮查看结果

## 🧮 计算模型

### 核心公式

#### 火球半径
```
R_fireball = 0.145 × Y^0.4  (km)
```
其中 Y 为当量 (kt)

#### 冲击波半径
```
R_blast = k × Y^(1/3)  (km)

k 值:
- 20 psi (重度): k = 0.28
- 5 psi (中度):  k = 0.60
- 2 psi (轻度):  k = 1.00
```

#### 热辐射半径
```
R_thermal = 1.9 × Y^0.41  (km)
```

#### 电离辐射半径
```
R_radiation = 1.2 × Y^0.19  (km)
```

#### EMP 影响半径
```
R_emp = base + Y × factor

地面爆炸: base = 10, factor = 0.1
空中爆炸: base = 30, factor = 0.2
高空爆炸: base = 100, factor = 0.5
```

### 伤亡估算模型

#### 有效人口密度
```
D_effective = D_base × (1 + U × T)

D_base: 基础人口密度
U: 城市化率
T: 时间系数 (夜间: -0.2, 白天: +0.3, 高峰: +0.5)
```

#### 伤亡计算
```
Deaths = Σ(Area_i × D_effective × M_i × F_shelter × F_building)

M_i: 各区域死亡率
F_shelter: 避难所系数 = 1 - (避难所覆盖率 × 0.8)
F_building: 建筑系数 = 1 - (建筑抗爆能力 × 0.5)
```

#### 医疗能力影响
```
Survivable_injuries = Total_injuries × Medical_capacity
Additional_deaths = Total_injuries × (1 - Medical_capacity) × 0.3
```

### 建筑类型参数

| 建筑类型 | 抗爆能力 | 防火能力 |
|----------|----------|----------|
| 木结构 | 0.30 | 0.20 |
| 砖结构 | 0.50 | 0.40 |
| 混凝土 | 0.70 | 0.60 |
| 现代建筑 | 0.60 | 0.50 |
| 抗震建筑 | 0.75 | 0.55 |
| 高层建筑 | 0.65 | 0.50 |
| 低层建筑 | 0.35 | 0.30 |
| 混合结构 | 0.50 | 0.45 |
