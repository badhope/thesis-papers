#### ✅ 子模块M1-1: EDGAR主索引构建

输入: SEC master.idx 2020-2024

  - FTP批量下载 quarterly master.idx
  - FORM_TYPE = '10-K' 精确过滤
  - 2020-11-01之后提交过滤
  - CIK去重与标准化

#### ✅ 子模块M1-2: 批量异步下载器
```
输入: sec_10k_index.csv
输出: 24,235份原始HTML文件

  - aiohttp 异步并发池 (并发=10)
  - SEC速率限制: 10请求/秒

  - 噪声过滤: 去除表格、图片、超链接、页脚
  - 提取准确率人工校验(100样本)
```

## ⚠️ SEC爬虫防封禁黄金守则

1. **User-Agent伪装**: 必须带真实邮箱，SEC要求
   ```python
   headers = {'User-Agent': 'Your Name your.email@university.edu'}
   ```

2. **速率严格控制**: 10请求/秒是硬限制，超过直接封禁IP

3. **重试策略**: 429状态码触发指数退避

4. **白天/夜间策略**: 夜间可以适当提高并发

5. **日志留存**: 每次请求详细日志，便于排查
